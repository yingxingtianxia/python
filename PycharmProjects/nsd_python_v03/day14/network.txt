# OSI参考模型：物理层、数据链路层、网络层、传输层、会话层、表示层、应用层
# 数据链路层的主要设备是交换机，主要的地址是mac地址
# 网络层主要设备是路由器，主要的地址是IP地址
# ipv4地址是32位２进制数，每８位换算成１０进制数，用小数点隔开，称作点分十进制
# dns服务器负责把好记的名字转换成不好记的ip地址
# 传层层重要的协议有tcp和udp，它们都有相应的端口号，协议加端口号标识了应用层的服务(协议)
# 通过ip地址可以找到主机，通过协议加端口号可以找到主机上提供的服务，比如web服务使用的是应用层的http协议，http协议对应的是传输层的tcp 80端口
# TCP即传输控制协议，它是面向连接的、可靠的协议
# UDP即用户数据报协议，它是非面向连接的、不可靠的协议
